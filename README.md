# BlogPostWebsite
Mini Blog Posting Website

This is just a mini prototype of a blog post website and contains the following functions:
-Register: New users can sign up by clicking the "register" button and entering their information, which will be stored in database.
-Login: Once registered, users can log in/sign in to their account, comment on posts, and read them.
-Logout: If the user wants to log out from his/her profile, there is a log-out button to do that.
-Contact: The "Contact" button will get hold of any issues or recommendations for admins of the website, and send it through email. Once admins read the message from the user, they will reach out to the developer, and also to the user back telling them that the problem has been addressed.
